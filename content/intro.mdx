# Introduction to Pumki UI

Welcome to Pumki UI! This introduction will help you get started with our comprehensive React component library.

## What is Pumki UI?

Pumki UI is a modern React component library designed to help developers ship stunning UIs quickly and efficiently. Instead of managing multiple libraries, Pumki UI provides everything you need in one cohesive package.

## Getting Started

Ready to start building? Let's get Pumki UI set up in your project.

### Installation

```bash copy
npm install pumki-ui
```

### Basic Usage

```tsx
import { Button } from 'pumki-ui'

export default function App() {
  return (
    <Button variant="primary">
      Hello, Pumki UI!
    </Button>
  )
}
```
