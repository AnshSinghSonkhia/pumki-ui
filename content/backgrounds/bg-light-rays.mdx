# Light Rays Background ðŸŒŒ

This component creates a mesmerizing light rays background effect, perfect for adding a dynamic and immersive atmosphere to your web applications. 

<Callout>The rays can be customized in terms of origin, color, speed, and more.</Callout>

<Preview 
  code={`'use client';

import LightRays from '@/components/pumki-ui/backgrounds/bg-light-rays'

export const LightRaysDemo = () => {
    return (
        <LightRays
            raysOrigin="top-center"
            raysColor="#00ffff"
            raysSpeed={1.5}
            lightSpread={0.8}
            rayLength={1.2}
            followMouse={true}
            mouseInfluence={0.1}
            noiseAmount={0.1}
            distortion={0.05}
            className="custom-rays"
        />
    );
};`}
>
