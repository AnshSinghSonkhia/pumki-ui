# Installation

## Installation Option 1

<Callout>Use without installation (Recommended ✅)</Callout>

No installation needed — add components directly with your package manager:

<Callout type="info">Example: Let's add "stats-carousel" component</Callout>

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']} defaultIndex="1">
  <Tabs.Tab>
    <CodeBlock language="bash" highlight="/pumki add/" showLineNumbers={false}>
      {`pnpm dlx pumki add stats-carousel`}
    </CodeBlock>
  </Tabs.Tab>
  <Tabs.Tab>
    <CodeBlock language="bash" highlight="/pumki add/" showLineNumbers={false}>
      {`npx pumki add stats-carousel`}
    </CodeBlock>
  </Tabs.Tab>
  <Tabs.Tab>
    <CodeBlock language="bash" highlight="/pumki add/" showLineNumbers={false}>
      {`yarn dlx pumki add stats-carousel`}
    </CodeBlock>
  </Tabs.Tab>
  <Tabs.Tab>
    <CodeBlock language="bash" highlight="/pumki add/" showLineNumbers={true}>
      {`# macOS/Linux
bunx pumki add stats-carousel

# or (works everywhere)
bun x pumki add stats-carousel

# Windows (PowerShell/CMD) — use this:
bun x pumki add stats-carousel`}
    </CodeBlock>
  </Tabs.Tab>
</Tabs>

<Callout type="important">Note: bunx is a symlink available on macOS/Linux. On Windows, use bun x.</Callout>

## Installation Option 2

<Callout>Global install (for convenience)</Callout>

<Callout type="important">If you want to avoid typing npx every time:</Callout> 

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']} defaultIndex="1">
  <Tabs.Tab>
    <CodeBlock language="bash" highlight="1" showLineNumbers={false}>
      {`pnpm add -g pumki`}
    </CodeBlock>
  </Tabs.Tab>
  <Tabs.Tab>
    <CodeBlock language="bash" highlight="1" showLineNumbers={false}>
      {`npm i -g pumki`}
    </CodeBlock>
  </Tabs.Tab>
  <Tabs.Tab>
    <CodeBlock language="bash" highlight="1" showLineNumbers={false}>
      {`yarn global add pumki`}
    </CodeBlock>
  </Tabs.Tab>
  <Tabs.Tab>
    <CodeBlock language="bash" highlight="1" showLineNumbers={false}>
      {`bun add -g pumki`}
    </CodeBlock>
  </Tabs.Tab>
</Tabs>

<Callout>Then you can add component like:</Callout>

<br />

<CodeBlock language="bash" highlight="1" showLineNumbers={false}>
{`pumki add stats-carousel`}
</CodeBlock>
