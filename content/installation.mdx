# Installation

<Callout>Install and configure PumkiUI Components</Callout>

To get started with **Pumki UI**, install the core packages into your React or Next.js project.

We recommend using `pnpm` for performance, but any package manager will work.

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']} defaultIndex="1">
  <Tabs.Tab>
    <CodeBlock language="bash" highlight="/pumki-ui/">
      {`pnpm add pumki-ui`}
    </CodeBlock>
  </Tabs.Tab>
  <Tabs.Tab>
    <CodeBlock language="bash" highlight="/pumki-ui/">
      {`npm install pumki-ui`}
    </CodeBlock>
  </Tabs.Tab>
  <Tabs.Tab>
    <CodeBlock language="bash" highlight="/pumki-ui/">
      {`yarn add pumki-ui`}
    </CodeBlock>
  </Tabs.Tab>
  <Tabs.Tab>
    <CodeBlock language="bash" highlight="/pumki-ui/">
      {`bun add pumki-ui`}
    </CodeBlock>
  </Tabs.Tab>
</Tabs>

## Optional: Tailwind CSS

Pumki UI works without Tailwind, but you can integrate it easily if you're already using Tailwind in your project.

If not already installed:

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']} defaultIndex="1">
  <Tabs.Tab>
    <CodeBlock language="bash">
      {`pnpm add -D tailwindcss postcss autoprefixer
npx tailwindcss init -p`}
    </CodeBlock>
  </Tabs.Tab>
  <Tabs.Tab>
    <CodeBlock language="bash">
      {`npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p`}
    </CodeBlock>
  </Tabs.Tab>
  <Tabs.Tab>
    <CodeBlock language="bash">
      {`yarn add -D tailwindcss postcss autoprefixer
npx tailwindcss init -p`}
    </CodeBlock>
  </Tabs.Tab>
  <Tabs.Tab>
    <CodeBlock language="bash">
      {`bun add -D tailwindcss postcss autoprefixer
npx tailwindcss init -p`}
    </CodeBlock>
  </Tabs.Tab>
</Tabs>

Then configure your `tailwind.config.js`:

<br />

<CodeBlock
  language="javascript"
  filename="tailwind.config.js"
  highlight="6"
>
{`/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    './app/**/*.{js,ts,jsx,tsx}',
    './components/**/*.{js,ts,jsx,tsx}',
    './node_modules/pumki-ui/**/*.{js,ts,jsx,tsx}',
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}`}
</CodeBlock>

<Callout type="info">Make sure your app is already set up with Tailwind before using Pumki UI components with utility classes.</Callout>
