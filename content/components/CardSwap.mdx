# Card Swap Animation

A 3D animated card stack that cycles through multiple cards with configurable depth, skew, distance, and animation easing.

<Preview
  code={`import CardSwap , { Card } from '@/components/pumki-ui/CardSwap'

export const CardSwapDemo = () => {
  return (
    <div className="relative w-full h-[500px] bg-gray-900 text-white flex items-center justify-center">
      <CardSwap width={300} height={400} cardDistance={50} verticalDistance={60} delay={3000} skewAmount={6} easing="elastic">
        <Card customClass="bg-pink-500 flex items-center justify-center text-xl">Card 1</Card>
        <Card customClass="bg-blue-500 flex items-center justify-center text-xl">Card 2</Card>
        <Card customClass="bg-green-500 flex items-center justify-center text-xl">Card 3</Card>
      </CardSwap>
    </div>
  );
};`}
>
  <CardSwapDemo />
</Preview>

## Usage

<br/>
<CodeBlock language="typescript">
{`import CardSwap , { Card } from '@/components/pumki-ui/CardSwap';`}
</CodeBlock>

<br/>
Wrap multiple `<Card>` elements inside `<CardSwap>` to create a rotating stack effect.

<CodeBlock language="typescript">
{`<CardSwap width={300} height={400} cardDistance={50} verticalDistance={60} delay={3000}>
    <Card customClass="bg-red-500">Card 1</Card>
    <Card customClass="bg-blue-500">Card 2</Card>
    <Card customClass="bg-green-500">Card 3</Card>
</CardSwap>`}
</CodeBlock>

### CardSwap Props

<PumkiTable>
  <table className="pumki-table">

        <th>Type</th>

      </tr>
    </thead>
    <tbody>
      <tr>
        <td><code>width</code></td>
        <td><code>number | string</code></td>
        <td><code>500</code></td>
        <td>Width of each card in pixels or CSS units.</td>
      </tr>
      <tr>
        <td><code>height</code></td>
        <td><code>number | string</code></td>
        <td><code>400</code></td>
        <td>Height of each card in pixels or CSS units.</td>
      </tr>
      <tr>
        <td><code>cardDistance</code></td>
        <td><code>number</code></td>
        <td><code>60</code></td>
        <td>Horizontal offset distance between cards in the stack.</td>
      </tr>
      <tr>
        <td><code>verticalDistance</code></td>
        <td><code>number</code></td>
        <td><code>70</code></td>
        <td>Vertical offset distance between cards in the stack.</td>
      </tr>
      <tr>
        <td><code>delay</code></td>
        <td><code>number</code></td>
        <td><code>5000</code></td>
        <td>Delay in milliseconds between card swaps.</td>
      </tr>
      <tr>
        <td><code>pauseOnHover</code></td>
        <td><code>boolean</code></td>
        <td><code>false</code></td>
        <td>Pauses the card swap animation when hovered.</td>
      </tr>
      <tr>
        <td><code>onCardClick</code></td>
        <td><code>(idx: number) =&gt; void</code></td>
        <td><code>-</code></td>
        <td>Callback triggered when a card is clicked, passing its index.</td>
      </tr>
      <tr>
        <td><code>skewAmount</code></td>
        <td><code>number</code></td>
        <td><code>6</code></td>
        <td>Skew value applied to cards for perspective effect.</td>
      </tr>
      <tr>
        <td><code>easing</code></td>
        <td><code>"linear" | "elastic"</code></td>
        <td><code>"elastic"</code></td>
        <td>Animation easing style for swaps.</td>
      </tr>
      <tr>
        <td><code>children</code></td>
        <td><code>ReactNode</code></td>
        <td><code>-</code></td>
        <td>List of <code>&lt;Card&gt;</code> elements to display in the swap animation.</td>
      </tr>
    </tbody>
  </table>
