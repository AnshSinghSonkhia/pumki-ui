# Show Code Preview to Users

A customizable code preview component for displaying code snippets with syntax highlighting.

<Preview 
  code={`import { Preview, ResponsivePreview } from '@/components/pumki-ui/preview'`}>
        <Preview
            title="Working Preview"
            code={`/* Code here is shown to users */
            `}
        >
        <LustrousText
            className="text-4xl font-semibold tracking-wide"
            text="Ansh Singh Sonkhia India"
            wave
            lustrousColor="#FFD700"
            color="#B8860B"
            duration={1.5}
        />
        </Preview>
</Preview>
