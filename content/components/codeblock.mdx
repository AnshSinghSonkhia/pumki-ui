# Code Block

A customizable code block component for displaying code snippets with syntax highlighting.



## Installation

## Usage

<br/>

<CodeBlock language="mdx" filename="example.mdx">
    {`
    import { CodeBlock } from '@/components/pumki-ui/codeblock';

    `}
</CodeBlock>
