# Text Rotator Animation

A dynamic text animation that smoothly rotates through words or phrases, adding energy and motion to your headlines, hero sections, and banners.

<Preview
  code={`import { RotatingText } from '@/components/pumki-ui/text/text-rotator'

  export const RotatingTextDemo = () => {
    return (
        <RotatingText
            className="text-4xl font-semibold"
            text={['Ansh', 'Singh', 'Sonkhia', 'India']}
        />
    );
};`}
>
    <RotatingText
      className="text-4xl font-semibold"
      text={['Ansh', 'Singh', 'Sonkhia', 'India']}
    />

</Preview>
