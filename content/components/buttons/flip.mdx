# Flip Button

A button that flips between two states with an accompanying animation.

<Preview 
  code={`import { FlipButton } from '@/components/pumki-ui/buttons/flip';

export const FlipButtonDemo = () => {
    return (
        <div className='flex gap-6'>
            <FlipButton frontText="Hello Pumki!" backText="Love you Pumki!" />
        </div>
    );
};`}
>
    <div className='flex gap-6'>
        <FlipButton frontText="Hello Pumki!" backText="Love you Pumki!" />
    </div>
    
</Preview>
