# List of commands I used while building this project:

```bash

npx create-next-app@latest              # Create Next.js app

npx shadcn@latest init                  # initialize shadcn 
# with `zinc` theme 

npx shadcn@latest add                   # to add all the shadcn components to my project

npm i next-view-transitions --force     # for smooth page transitions and visual effects during navigation within a Next.js application

npm i lucide-react

npm install react-use-measure

npm install react@18.2.0 react-dom@18.2.0   
```
